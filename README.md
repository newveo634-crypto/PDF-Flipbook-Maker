# 📚 Interactive PDF Flipbook Maker

A professional web application that converts static PDF documents into interactive, 3D animated flipbooks. Perfect for graphic designers, digital magazines, and product catalogs.

## 🎨 Why Use This?
As a graphic designer, I created this tool to help clients showcase their work in a more engaging way. Instead of a standard scrolling PDF, this provides a realistic "page-turning" experience directly in the browser.

## 🚀 Features
* **Realistic 3D Animation:** Smooth page-flip effects powered by StPageFlip.
* **Instant Conversion:** Upload a PDF and view it as a book immediately.
* **Responsive Design:** Works on desktops, tablets, and mobile devices.
* **Custom Branding:** Clean interface that keeps the focus on the content.

## 🛠️ Built With
* **Frontend:** HTML5, CSS3, JavaScript (PDF.js, StPageFlip)
* **Backend:** Python (Flask)
* **Hosting:** PythonAnywhere

## 📂 Project Structure
- `app.py`: The Python backend handling file uploads and routing.
- `templates/`: Contains the user interface and flipbook engine.
- `uploads/`: Secure storage for converted PDF files.
- `requirements.txt`: List of Python dependencies for the server.
